ZAP API UI

Component: reports

Action: generate

Generate a report with the supplied parameters.
Output Format
apikey*
Form Method
title*Report Title
template*Report Template
themeReport Theme
descriptionReport Description
contextsThe name of the contexts to be included in the report, separated by '|'.
sitesThe site URLs that should be included in the report, separated by '|'.
sectionsThe report sections that should be included, separated by '|'.
includedConfidencesConfidences that should be included in the report, separated by '|'. Accepted values are "False Positive", "Low", "Medium", "High", and "Confirmed".
includedRisksRisks that should be included in the report, separated by '|'. Accepted values are "Informational", "Low", "Medium", and "High".
reportFileNameThe file name of the generated report. This value overrides the reportFileNamePattern parameter.
reportFileNamePatternReport File Name Pattern.
reportDirPath to directory in which the generated report should be placed.
displayDisplay the generated report. Either "true" or "false".
